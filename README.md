#  AI Chess Game

##  Overview

This project is a Python-based AI Chess Game that allows a user to play against a computer opponent. The system uses the Minimax algorithm to make intelligent decisions, enabling the AI to evaluate possible moves and choose the optimal strategy.

---

##  Features

*  Fully playable chess game
*  AI opponent using Minimax algorithm
*  Decision-making based on game tree evaluation
*  Object-oriented design for pieces and board
*  Interactive gameplay with graphical interface

---

##  Concepts Used

* Minimax Algorithm (Game AI)
* Game Tree Search
* Heuristic Evaluation
* Object-Oriented Programming (OOP)
* State Space Representation

---

##  Project Structure

* `main.py` → Entry point of the game
* `Board.py` → Handles board state and logic
* `ChessPiece.py` → Defines piece behavior
* `Computer.py` → AI logic for move selection
* `graphics.py` → UI and rendering
* `minimax_tree` → Minimax implementation

---

##  How to Run

1. Clone the repository
2. Open the folder in VS Code or any Python IDE
3. Make sure Python is installed
4. Run:

```bash
python main.py
```

---

##  Gameplay

* The player competes against an AI opponent
* The AI evaluates moves using Minimax
* The game continues until checkmate or game over
